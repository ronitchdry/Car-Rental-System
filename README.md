Car Rental System – Java OOP Project

A console-based car rental system built using Java and Object-Oriented Programming principles.  
The project demonstrates encapsulation, class interaction, and clean object-oriented design.

Features:
- Rent and return cars
- Track car availability
- Price calculation based on rental days
- Customer management
- Menu-driven console interface

Technologies:
- Java
- OOP (Encapsulation, Classes, Objects)
- Java Collections
- Console I/O
